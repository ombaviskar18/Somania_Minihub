## ğŸš€ Getting Started

1. **Clone the repository**
```bash
git clone https://github.com/ombaviskar18/Somnia_Minihub.git
cd Somnia_Minihub
```

2. **Install dependencies**
```bash
npm install
# or

```

3. **Set up environment variables**
